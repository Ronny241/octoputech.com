Web Development Tags Reference

HTML Tags

  • <a>: Defines a hyperlink.
  • <abbr>: Defines an abbreviation or acronym.
  • <address>: Defines contact information.
  • <area>: Defines an area inside an image map.
  • <article>: Defines independent content.
  • <aside>: Defines content aside from the main content.
  • <audio>: Embeds sound content.
  • <b>: Defines bold text.
  • <base>: Specifies the base URL for all relative URLs.
  • <bdi>: Isolates text directionally.
  • <bdo>: Overrides the current text direction.
  • <blockquote>: Defines a block of quoted text.
  • <body>: Defines the document's body.
  • <br>: Inserts a single line break.
  • <button>: Defines a clickable button.
  • <canvas>: Used for graphics drawing.
  • <caption>: Defines a table caption.
  • <cite>: Defines a citation.
  • <code>: Defines computer code text.
  • <col>: Specifies column properties for each column.
  • <colgroup>: Specifies a group of columns.
  • <data>: Links content to a machine-readable value.
  • <datalist>: Specifies a list of pre-defined options.
  • <dd>: Describes a term/name in a description list.
  • <del>: Defines deleted text.
  • <details>: Defines additional details.
  • <dfn>: Specifies a term that is being defined.
  • <dialog>: Defines a dialog box.
  • <div>: Defines a section in a document.
  • <dl>: Defines a description list.
  • <dt>: Defines a term/name in a description list.
  • <em>: Defines emphasized text.
  • <embed>: Embeds external content.
  • <fieldset>: Groups related elements in a form.
  • <figcaption>: Defines a caption for a figure.
  • <figure>: Groups media content.
  • <footer>: Defines a footer.
  • <form>: Defines an HTML form.
  • <h1> to <h6>: Define HTML headings.
  • <head>: Contains meta-information about the document.
  • <header>: Defines a header.
  • <hgroup>: Groups headings.
  • <hr>: Defines a thematic break.
  • <html>: Defines the root element.
  • <i>: Defines italicized text.
  • <iframe>: Defines an inline frame.
  • <img>: Embeds an image.
  • <input>: Defines an input field.
  • <ins>: Defines inserted text.
  • <kbd>: Defines keyboard input.
  • <label>: Defines a label for an input element.
  • <legend>: Defines a caption for a <fieldset> element.
  • <li>: Defines a list item.
  • <link>: Defines a relationship between a document and an external resource.
  • <main>: Specifies the main content of the document.
  • <map>: Defines an image map.
  • <mark>: Highlights text.
  • <meta>: Defines metadata.
  • <meter>: Defines a scalar measurement.
  • <nav>: Defines navigation links.
  • <noscript>: Defines alternate content for users who don't support scripts.
  • <object>: Embeds an external resource.
  • <ol>: Defines an ordered list.
  • <option>: Defines an option in a drop-down list.
  • <output>: Represents the result of a calculation.

CSS Properties

  • color: Sets the color of text.
  • background-color: Sets the background color of an element.
  • font-size: Sets the size of the font.
  • margin: Sets the space around an element.
  • padding: Sets the space inside an element.
  • border: Sets the border of an element.
  • display: Specifies the display behavior of an element.
  • flex: Defines how a flex item will grow or shrink.
  • grid-template-columns: Defines the column structure of a grid container.
  • align-items: Aligns items vertically in a flex or grid container.
  • justify-content: Aligns items horizontally in a flex or grid container.
  • position: Specifies the type of positioning for an element.
  • top: Specifies the top position of an element.
  • left: Specifies the left position of an element.
  • right: Specifies the right position of an element.
  • bottom: Specifies the bottom position of an element.
  • z-index: Specifies the stack order of an element.
  • box-shadow: Adds shadow to an element's box.
  • text-align: Specifies the horizontal alignment of text.
  • line-height: Sets the height of lines of text.
  • opacity: Sets the transparency level of an element.
  • transform: Applies transformations like rotate, scale, and skew.
  • transition: Adds animation between property changes.
  • animation: Specifies animations for elements.
  • clip-path: Defines a clipping region for an element.
  • overflow: Controls how content is handled when it's larger than its container.

JavaScript Methods

  • addEventListener(): Attaches an event handler to an element.
  • getElementById(): Selects an element by its ID.
  • querySelector(): Selects the first element matching a CSS selector.
  • querySelectorAll(): Selects all elements matching a CSS selector.
  • createElement(): Creates a new HTML element.
  • appendChild(): Appends a child element to a parent element.
  • removeChild(): Removes a child element from a parent element.
  • setAttribute(): Sets an attribute for an element.
  • getAttribute(): Gets the value of an attribute.
  • add(): Adds an item to a Set or a class to an element.
  • remove(): Removes an item from a Set or a class from an element.
  • replace(): Replaces a substring in a string.
  • toString(): Converts a value to a string.
  • parseInt(): Parses a string into an integer.
  • parseFloat(): Parses a string into a floating-point number.
  • JSON.stringify(): Converts an object to a JSON string.
  • JSON.parse(): Parses a JSON string into an object.
  • setTimeout(): Executes a function after a specified delay.
  • setInterval(): Repeats a function at specified intervals.
  • fetch(): Performs an HTTP request.
  • map(): Creates a new array by applying a function to each element.
  • filter(): Creates a new array with elements that meet a condition.
  • reduce(): Reduces an array to a single value.
  • sort(): Sorts an array in place.
  • reverse(): Reverses the order of elements in an array.
  • includes(): Checks if an array includes a specified element.
  • push(): Adds an element to the end of an array.
  • pop(): Removes the last element from an array.
  • shift(): Removes the first element from an array.
  • unshift(): Adds an element to the beginning of an array.
